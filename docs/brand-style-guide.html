01 — The System

One family, one voice

Every Boulder BioMed company shares one typeface (Lato), one logo system, one color palette, and one website architecture. Two things are non-negotiable across all sites: the Lato font and the logo placement. Color has limited flexibility within the approved palette below.

08 — Usage

Do & don't

✓ DO
  • Use the inline lockup: Boulder ｜ descriptor, descriptor at 0.56× in Lato Regular.
  • Descriptor in Boulder Blue on light, Sky Blue on dark, white on color tiles.
  • Set every site in Lato, loaded from Google Fonts.
  • Use Deep Navy for footers and dark sections; Green for accents.
  • Keep the vertical divider at ~1× the height of "Boulder".
✕ DON'T
  • Don't bold the descriptor or set it the same size as "Boulder".
  • Don't use system fonts, Open Sans, or any non-Lato face.
  • Don't reintroduce teal; don't recolor "Boulder" to blue.
  • Don't stretch, skew, shadow, or outline the lockup.
  • Don't mix a light-background logo onto a dark field.

11 — Implementation

Design tokens

Drop these into every site (CSS variables or Tailwind theme) so color and type are locked across the family.

:root{
  --boulder-blue:    #2968A0;  /* primary · links · buttons · descriptor */
  --navy:            #0B2545;  /* headings · dark sections · footer      */
  --sky-blue:        #5B8FC7;  /* descriptor on dark backgrounds         */
  --signature-green: #75D69C;  /* accent · stat numbers · highlights     */
  --slate:           #414856;  /* "Boulder" wordmark on light            */
  --ink:             #2A2A2A;  /* body text                              */
  --boulder-gray:    #C9CFD6;  /* "Boulder" on dark                      */
  --cloud:           #F4F6F8;  /* light section background               */
  --radius:          6px;      /* buttons / cards                        */
  --font:            'Lato', -apple-system, sans-serif;
}
